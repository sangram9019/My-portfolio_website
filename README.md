# Sangram's Portfolio

This is my personal portfolio website built with React and Vite.

## Project Structure

This project uses the standard Vite template:

- `src/` contains the source code.
- `public/` contains static assets like the resume.
- `dist/` contains the production build.

## Development

To run locally:
```bash
npm run dev
```

## Deployment

This project is deployed using Vercel/Netlify.
